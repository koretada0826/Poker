# 3時間でポーカー実卓デビュー（学習ゲーム）

完全初心者が、3時間でアミューズメントポーカーの実卓に座っても困らないレベルになることを目指す、実践型の学習Webアプリです。

> ⚠️ ギャンブル・金銭賭博を推奨するものではありません。あくまでゲームとしてのポーカー学習用です。「必ず勝てる」「稼げる」ことを保証するものではありません。

## できること

- トランプ超基礎（マーク・絵札・カードの強さ）
- ポーカーの役 10種類
- テキサスホールデムの流れ（プリ/フロップ/ターン/リバー/ショーダウン）
- アクション（チェック・ベット・コール・レイズ・フォールド）の使い分け
- 初心者向けプリフロップ判断（強い手・状況次第・降りるべき手）
- フロップ後の危険サイン（同マーク3枚、連続数字）
- CPU相手のヘッズアップ実戦シミュレーション
- アミューズメントポーカーのマナー
- 卒業試験（合格で「実卓デビューOK」称号）

## 起動方法

```bash
# 1. 依存をインストール
npm install

# 2. 開発サーバを起動
npm run dev
# → http://localhost:5173 が自動で開きます

# 3. （任意）本番ビルド
npm run build
npm run preview
```

Node.js 18 以降を推奨。

## 使い方

ホーム画面のステップに沿って、上から順に進めるのが推奨です。

1. **STEP 1〜2**：レッスンでトランプ・役を覚える
2. **練習**：役判定トレーニングで反復
3. **STEP 3〜4**：流れとアクションを覚える
4. **STEP 5**：初心者戦略
5. **STEP 6**：実戦シミュレーションで判断練習
6. **FINAL**：卒業試験

進捗は `localStorage` に自動保存されます。

## 技術スタック

- Vite + React + TypeScript
- Tailwind CSS
- 状態管理：React Hooks + localStorage（外部DB不要）

## 構成

```
src/
├── main.tsx              # エントリ
├── App.tsx               # ルーティング
├── index.css             # Tailwind + カスタムCSS（カードUI）
├── types.ts              # 型定義
├── components/           # Card/CardRow/QuizRunner/Feedback/Header
├── screens/              # Home/Lesson/Practice/Simulation/Exam
├── lib/
│   ├── deck.ts           # デッキ生成・シャッフル
│   ├── handEvaluator.ts  # 5〜7枚から最強役を判定
│   ├── quizGenerator.ts  # クイズ自動生成
│   ├── simulation.ts     # ヘッズアップ簡易シミュレーション
│   └── storage.ts        # 進捗保存
└── data/
    └── lessons.ts        # レッスン本文
```

## 設計ポリシー

- **実践中心**：説明 → クイズ → 実践 → フィードバック → 次へ
- **小学生にも分かる日本語**：専門用語は必ず日本語で説明
- **5秒チェック**：①自分の手は強い？ ②場の危険は？ ③相手は強そう？ ④払う価値は？ ⑤迷ったら降りる
- **「結果」より「判断」を評価**：正しい判断を続けることを称賛
